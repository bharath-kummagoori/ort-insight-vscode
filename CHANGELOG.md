# Changelog

All notable changes to the "ORT Insight" extension will be documented in this file.

## [0.1.0] - 2024-01-01

### Added
- Initial release of ORT Insight
- ORT Analyzer integration
- Dependency tree view with license information
- License compliance dashboard with visualizations
- SBOM generation (CycloneDX and SPDX formats)
- ORT Advisor integration for vulnerability checking
- Inline diagnostics for problematic licenses
- Status bar integration showing compliance status
- Configurable license risk classification
- Support for all ORT-supported package managers
- Output channel for ORT command execution
- Workspace folder support
